# tetris-raylib-project
Tetris game recreation in C using the Raylib framework/library.

Current features:

- Piece spawning system
- Piece rotation 
- Vertical movement (by gravity and player input) 
- Piece placing
- Wall collisions

Missing features:

- Main menu
- Settings menu
- Other 6 piece types
- Default Tetris random piece spawning
- Horizontal collisions with placed pieces
- Scoring system
- "Game Over" possibility
